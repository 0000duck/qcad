QCAD 3.1

This is QCAD version 3.1.

QCAD is an application for computer aided drafting in two dimensions (2d).
With QCAD you can create technical drawings such as plans for buildings, 
interiors, mechanical parts or schematics and diagrams. QCAD works on 
Windows, Mac OS X and many Linux and Unix Systems. The source code of 
the QCAD Community Edition is released under the GPL (Open Source).
