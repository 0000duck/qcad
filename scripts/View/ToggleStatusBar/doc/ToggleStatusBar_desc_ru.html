The status bar at the bottom of the application window shows various information about the current status of QCAD.
At the left, the coordinate display shows the absolute and relative position of the mouse cursor as Cartesian and polar coordinates.
The mouse widget in the middle shows information about the current function of the left and right mouse buttons.
The selection widget at the right shows the number of entities that are currently selected.
The status bar can be enabled / disabled using this tool.
