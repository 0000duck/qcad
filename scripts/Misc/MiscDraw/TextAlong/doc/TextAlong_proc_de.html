1. Nach dem Start des Werkzeuges erscheint der Textdialog, in dem Sie die Schriftart auswählen und den zu erstellenden Text eingeben können.
  2. Wählen Sie im oberen linken Bereich des Dialogs den Schriftnamen, die Texthöhe und die Textattribute.
    CAD-Schriften, die sich aus Linien und Bögen zusammensetzen, sind oben in der Schriftliste aufgelistet, TrueType-Schriften weiter unten. Um den normalen Zeilenabstand zu wählen, geben Sie einen Zeilenabstandsfaktor von "1" ein. Sie können den Zeilenabstand vergrössern oder verkleinern, indem Sie einen alternativen Wert eingeben. Der eingegebene Wert ist ein Faktor relativ zur Voreinstellung (1).
  3. Geben Sie den Drehwinkel für den Text ein. Normale horizontale Texte haben einen Winkel von 0 Grad und Texte, die von rechts lesbar sind 90 Grad.
  4. Wählen Sie die Ausrichtung des Textes.
  5. Aktivieren Sie das Kontrollkästchen "Einfacher Text", um einen einzeiligen Text ohne erweiterte Formatierung zu erstellen.
  6. Geben Sie den zu erstellenden Text in das grosse Textfeld ein.
    Mit den verschiedenen Werkzeugen oberhalb der Box können Sie den eingegebenen Text weiter formatieren oder Texte aus der Zwischenablage einfügen.
    Beachten Sie, dass die von Ihnen gewählte Schriftart möglicherweise nicht die von Ihnen eingefügten Zeichen enthält. In diesem Fall werden die Zeichen in der Zeichnung nicht angezeigt.
  7. Klicken Sie auf "OK", um den Textdialog zu verlassen.
  8. Geben Sie mit der Maus die Position des Textes an oder geben Sie in der Befehlszeile eine Koordinate ein.
  9. Häufig möchten Benutzer eine Reihe ähnlicher Texte anlegen. Aus diesem Grund wird das Textwerkzeug nach dem Erstellen des ersten Textes nicht beendet. Sie können den Text und den Textwinkel in der Optionenwerkzeugleiste ändern und so weitere Texte mit ähnlichen Einstellungen anlegen.
  10. Klicken Sie mit der rechten Maustaste oder drücken Sie Escapetaste, um das Textwerkzeug zu stoppen.
